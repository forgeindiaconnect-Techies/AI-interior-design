# Project Summary

## Goal
AI Interior Design & Marketplace platform (React + Vite frontend, Express/MongoDB backend, dual data: real DB + mock fallback).

## Constraints & Preferences
- Frontend: React, Vite, TailwindCSS, Axios, React Router, jsPDF, lucide-react
- Backend: Express, Mongoose, JWT, bcrypt, Razorpay
- Data strategy: real DB is primary; mock data arrays in both frontend and backend as fallback
- Mock IDs: `ord_*`, `mock_*`, `mkt_*`, `del_*`, `txn_*`, `v_*`, `sv_*` (not real MongoDB ObjectIds)
- Hosted on Render (free tier, spins down after inactivity)
- All API routes prefixed with `/api`
- Proxied via Vite config (`http://localhost:5000`)
